# 🪟 README WINDOWS - DÉMARRAGE RAPIDE

**Pour Windows 10 / Windows 11**

---

## ⚡ Démarrage Rapide (30 secondes)

### 1️⃣ Installez Python
- Téléchargez: https://www.python.org/downloads/
- **IMPORTANT:** Cochez "Add Python to PATH"
- Installez

### 2️⃣ Double-cliquez sur `launcher.bat`
```
launcher.bat  ← Cliquez 2 fois dessus
```

### 3️⃣ Choisissez votre option
```
1 = Interface Graphique
2 = Détection de Mains
3 = Détection Optimisée (Nouveau!)
```

**C'est tout!** 🎉

---

```
launcher.bat              ← Menu interactif (CLIQUEZ ICI EN PREMIER)
start_gui.bat             ← Lancer interface directement
```

## 📞 Problèmes?

**Python non reconnu?**
```
→ Réinstallez en cochant "Add Python to PATH"
→ Redémarrez Windows
```

**Caméra ne fonctionne pas?**
```
→ Vérifiez Paramètres → Confidentialité → Caméra
→ Cochez "Accès à la caméra"
```

**Erreur de dépendances?**
```
Ouvrez CMD et tapez:
pip install --upgrade opencv-python mediapipe
```
