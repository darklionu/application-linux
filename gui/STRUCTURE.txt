🚀 COMMANDES RAPIDES

Compiler:
  cd gui && make all

Nettoyer:
  cd gui && make clean

Setup:
  ./gui/setup.sh

Compiler un exemple:
  cd gui/src
  g++ -std=c++17 -I../include main_gui_example.cpp \
      gui_agent_panel.cpp gui_button.cpp -o example

💡 PROCHAINES ÉTAPES

1. Lire: gui/DÉMARRAGE.md
2. Choisir: SFML, Qt5, ou GTK3
3. Installer: sudo apt-get install libsfml-dev (recommandé)
4. Créer: gui/src/gui_window_sfml.cpp (implémentation)
5. Compiler: g++ ... -lsfml-graphics -lsfml-window -lsfml-system
6. Tester: ./gui_app
7. Intégrer: Lier à l'application principale
