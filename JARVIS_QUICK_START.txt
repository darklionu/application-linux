💡 PERSONNALISER VOTRE INTERFACE
─────────────────────────────────────────────────────────────────────────────

1. Ajouter un agent:
   Modifiez gui/src/main_jarvis.cpp et ajoutez:
   
   system.addAvailableAgent(
       "Nom Agent",
       "type_agent",
       "./chemin/agent",
       true
   );

2. Modifier le menu:
   Éditez gui/include/jarvis_interface.h et modifiez:
   
   void displayBanner() { ... }
   void processCommand() { ... }

3. Ajouter une commande:
   Complétez processCommand():
   
   if (operation == "ma_commande") {
       // Votre logique
   }
