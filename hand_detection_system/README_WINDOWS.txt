DÉMARRAGE RAPIDE (< 2 minutes):

1. Télécharger Python
   Site: https://www.python.org/downloads/
   ✅ IMPORTANT: Cocher "Add Python to PATH"

2. Vérifier Python
   Ouvrir CMD et taper:
   python --version

3. Installer les dépendances
   Double-cliquer sur:
   install_windows.bat

4. Lancer l'application
   Double-cliquer sur cmd et taper:
   python main_windows.py

================================================================================

FICHIERS DISPONIBLES:

1. windows_ultra_light.py
   - Simple, sans dépendances
   - Menu interactif
   - Usage: python windows_ultra_light.py

2. main_windows.py (RECOMMANDÉ)
   - Application complète
   - Avec OpenCV
   - Usage: python main_windows.py

3. main.py
   - Complète avec MediaPipe ML
   - Meilleure qualité
   - Usage: python main.py

================================================================================

SCRIPTS D'INSTALLATION:

1. install_windows.bat
   - Double-cliquez pour installer
   - Installe automatiquement OpenCV et MediaPipe
   - Recommandé pour CMD

2. install_windows.ps1
   - PowerShell script
   - Same functionality as .bat
   - Usage: .\install_windows.ps1

================================================================================

DÉPANNAGE WINDOWS:

❌ "Python not recognized"
   → Réinstaller Python avec "Add to PATH"

❌ "Camera not found"
   → Vérifier dans Paramètres → Confidentialité → Caméra

❌ "import cv2 failed"
   → Exécuter: pip install opencv-python

❌ "Permission denied"
   → Exécuter en tant qu'administrateur
